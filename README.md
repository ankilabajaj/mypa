# MyPA

MyPA is an AI-powered productivity assistant designed to help users take action before missing important deadlines.

Unlike traditional reminder apps that only send notifications, MyPA helps users prioritize tasks, identify what to work on next, and break large goals into manageable steps.

## Features

* Task creation and management
* Deadline tracking
* Priority levels (High, Medium, Low)
* Automatic task sorting by urgency
* Status indicators (Overdue, Due Today, This Week, Upcoming)
* Today's Focus recommendation
* Local storage persistence

## Future AI Features

* Intelligent task prioritization
* AI-generated task breakdowns
* Personalized daily plans
* Productivity recommendations
* Deadline risk prediction

## Built With

* React
* Vite
* JavaScript
* Google AI Studio (planned integration)
* Gemini API (planned integration)

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
